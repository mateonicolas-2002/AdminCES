Funcionalidades principales:
-Iniciar sesión
Permite al usuario acceder al sistema si ya tiene una cuenta creada.
Datos que utiliza:
Email
Contraseña
-Crear cuenta tester
Permite a un administrador crear nuevos usuarios
Datos que utiliza:
Nombre
Apellido
Email
País de nacimiento
Contraseña
Tester Junior/Tester Senior/TesterLíder
-Reiniciar contraseña
Permite recuperar o cambiar la contraseña
Datos que utiliza:
Email
Contraseña
Repetir contraseña
-Crear cuenta administrador
Permite crear una nueva cuenta de usuario
Datos que utiliza:
Nombre
Apellido
Email
Contraseña
Repetir contraseña
País nacimiento
-Ver usuarios
-Eliminar usuarios
Permite borrar usuarios registrados del sistema
Datos que utiliza:
selección del usuario a eliminar
-Ver perfil
Permite visualizar y editar los datos del usuario
Datos que utiliza:
Nombre
Apellido
Email
País
Perfil/rol
-Cerrar sesión
Permite salir de la cuenta actual del usuario
Datos que utiliza:
No requiere ingreso de datos
Comportamiento adicional:
El sistema muestra una ventana de confirmación antes de cerrar la sesión (“Sí” / “Cancelar”)
-Volver al menú principal
Permite regresar a la pantalla principal del sistema
Datos que utiliza:
No requiere datos
-Crear cuenta!
Te lleva a "Crear cuenta administrador"
-Olvide contraseña
Te lleva a "Reiniciar contraseña"

-Ya tengo cuenta!
Te lleva a "Iniciar sesión administrador"
